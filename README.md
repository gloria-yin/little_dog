# 🐶 小狗预设使用手册

一个可视化的文风 + 思维链速查本，专门整理给用 AI 写小说 / 跑RP的人看。

写到一半不知道该用什么文风？
COT 套模板套得心累，每个都长得差不多分不清？
想看看不同思维链到底出来是什么效果、啥场景该用哪个？

这个手册就是为了解决这些问题——
把所有文风（目前 128 条）和思维链（目前 73 条）全部整理成可筛选、可检索、可对比的卡片式文档，直接网页里翻。

## 🗝️ 核心功能

### 📖 128 条文风速查

按 **ID 编号** 整理的文风库，每条包含：

* **文风名称 + ID**（比如 `0013 · 晋江温情`）
* **NSFW 标识**（♥ 红心）
* **对应预设**：留光 / 行间 / 小狗 / 冬日小狗 / 众生相
* **使用方向**：都市纯爱、古风权谋、硬核写实……
* **参考作家**：汪曾祺、张爱玲、王家卫、Sally Rooney……
* **特点词 + 特点介绍**：五句话讲清楚这个文风是什么感觉
* **文风示例**：同一个"久别重逢"场景在不同文风下会写成什么样

> 所有示例都是同一个场景的变奏，方便横向对比——这是这本手册最爽的地方，你能直观看到"工笔写意"和"沉默废墟"写同一件事到底差在哪。

### 🧠 73 条思维链（COT）速查

每条 COT 包含：

* **名称 + ID**（比如 `0039 · 推进（快速）`）
* **对应预设 + 条目长短**（短 / 中 / 长 / 极长，对应的 token 消耗一目了然）
* **COT 特点词**
* **核心区别与特色**：一句话说清楚这条 COT 在干什么
* **建议应用场景**：什么时候该用它
* **核心区别 / 步骤**：COT 里的关键机制列表
* **场景展示**：同一个"冲突离开"和"撒娇亲近"场景，这条 COT 会怎么引导 AI 处理

> COT 部分的场景展示是神来之笔——直接看同一个情境下"叙事账本驱动"和"慢热拉扯"会引导出多么不同的剧情走向，比对着看比单纯读介绍高效十倍。

### 🔍 筛选 & 检索

顶部筛选栏（可折叠）：

* **搜索**：按标题搜 / 按特点搜（会扫描特点词 + 介绍 + 应用场景）
* **预设标签筛选**：
  * 文风模块支持**多选**（留光 + 小狗 + NSFW 可叠加筛选）
  * COT 模块支持**单选**（一次只看一个预设的 COT）

想找"小狗预设下的所有 NSFW 文风"？两个标签一点就行。

### 📚 侧边栏目录

左侧有个可折叠的目录（点左边那个小浮动按钮打开），
点击任意条目直接跳转到对应卡片，配合筛选用特别方便定位。

目录会根据当前在哪个 Tab（文风 / COT）自动切换内容。

### 🎨 视觉风格

整本手册做成了**复古纸张风**：
* 泛黄底色 + 微纹理
* 衬线字体 + 手写感标题
* 每张卡片都有四角装饰和轻微的内阴影
* NSFW 标识用红心脉动
* "冲突离开"和"撒娇亲近"两个场景盒子用了不同的色调区分
